# 自我介绍

我是一名拥有 5 年经验的前端工程师，目前在腾讯负责云控制台的架构设计。我的核心竞争力在前端架构和性能优化方向 —— 在腾讯主导了微前端架构落地，在字节跳动做过大型中后台的性能优化。

技术栈方面，React 和 Vue 都有深入使用经验，最近两年主要在 React + TypeScript 技术栈深耕。除了业务开发，我喜欢做开源项目，维护了一个 2.3k Stars 的电子书阅读器和一个 5000+ 周活的开发者工具扩展。

我是一个偏产品思维的前端，不只是关心技术实现，更关心技术方案对业务目标的支撑。在字节的时候主动推动动态表单引擎的开发，就是发现运营配置效率是个真实痛点。

目前在看新机会，主要关注几个方向：
1. 能接触到底层架构设计的机会
2. 产品技术氛围好的团队（工具类/SaaS 优先）
3. 有技术成长空间，不只是做业务迭代

不喜欢：
- 996 文化严重的公司
- 纯外包性质的开发工作
- 技术决策完全由后端主导的团队
