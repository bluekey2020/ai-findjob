- 技术栈有相似之处（都是React+TypeScript体系）

**具体操作：**

1. **通过腾讯前同事寻找**
   ```
   询问CSIG/其他BG的同事：
   "你知道有谁从腾讯跳去了微众吗？我想了解下那边的情况。"
   ```

2. **通过企业微信群/内部论坛**
   - 腾讯内部有离职员工群（"鹅厂校友群"类）
   - KM平台可能有"为什么离开腾讯去微众"的经验分享
   - 搜索"微众"关键词在内部通讯录中

3. **腾讯→微众的信任背书**
   ```
   腾讯在职/前员工的身份天然有信任加成：
   "我也是腾讯出来的" = 同一生态系统 = 天然亲近感
   ```

### 6.1 微众 vs 互联网大厂的核心差异

| 维度 | 互联网大厂（腾讯/字节） | 微众银行 | 对张三的影响 |

| 技术栈 | 前沿（React 18/Vite等） | 中等偏高 | 🟡 技术更新可能慢半拍 |
