## 一、2025-2026 深圳前端招聘市场总览

### 1.1 供需关系

| 维度 | 数据 |
|------|------|
| 高级前端工程师供需比 | 约 1:3（中等紧缺） |
| 前端架构师供需比 | 约 1:1.5（供不应求） |
| 深圳前端岗位同比增长 | 约 8%（2025→2026） |
| 深圳求职人数环比增速 | 42.1%（全国领跑） |
| AI相关前端岗位溢价 | 30%-50% |

### 1.2 热门技能需求

- **必会三件套**：React 18+、TypeScript、微前端架构
- **高溢价技能**：AI/LLM前端集成、Node.js全栈、WebAssembly、跨端方案（Flutter/鸿蒙）
- **新兴方向**：AI Copilot前端集成、Agent UI设计模式、D2C（Design to Code）
- **传统加分项**：组件库/设计系统建设、前端性能优化、构建工具链（Vite/Turbopack/Rspack）

### 1.3 薪资基准（深圳，月薪）

| 级别 | P25 | P50 | P75 | P90 |
|------|-----|-----|-----|-----|
| 高级前端工程师 | 28K | 35K | 42K | 50K |
| 前端架构师 | 38K | 45K | 55K | 65K |

> 张三当前预期 35K-45K 偏保守，基于腾讯+字节+Shopee大厂背景，建议上浮至 **38K-52K** 作为锚定区间。

## 二、目标公司逐个 HR 情报

### 2.1 腾讯（当前雇主）

#### 招聘流程
- **轮次**：社招一般 3轮技术面 + 1轮HR面（高级岗位可能追加交叉面）
- **一面（60-70min）**：项目深挖（STAR法则）+ JS核心（闭包/原型链/事件循环）+ 手写代码
- **二面（50-60min）**：框架原理（React Fiber/Vue3响应式）+ 网络/浏览器原理 + 性能优化设计
- **三面（30-60min）**：系统设计 + 架构思维 + 技术选型
- **HR面（30min）**：职业规划 + 协作能力 + 薪资谈判 + 文化匹配
- **2025年新增**：AI面试环节（通过"面呗"工具在线编程）

#### 薪资结构
| 组件 | 详情 |
|------|------|
| Base月薪 | T9：30K-50K / T10：45K-70K |
| 年终奖 | 3-6个月（核心业务线可达10-20个月） |
| 股票（RSU） | T9起：500-800股/年（约15-25万/年），4年分批归属 |
| 标准薪制 | 14薪（实际16-20薪） |
| 总包参考（T9-T10） | 60万-140万/年 |

#### 隐形门槛
- **学历**：社招本科起步，985/211在简历池中有优势但非硬性
- **年龄**：30-35岁属于黄金期，T10以上竞争激烈
- **跳槽频次**：2-3年一跳正常，5年3跳需解释
- **内部活水**：AI相关部门（混元/元宝/微信电商）仅需在岗满3个月即可申请

#### HC情况（2025-2026）
- 腾讯2025年秋招释放180人（含前后端/算法），HC相对充裕
- CSIG（云）经历2023-2024组织调整后趋于稳定
- 微信生态（视频号/小程序/支付海外）持续扩张
- AI方向（混元大模型）是HC倾斜重点

#### 内推价值
- **极高**：内推简历优先查看，内部员工可查询面试反馈
- 活水转岗是最短路径——张三当前在腾讯，直接活水到WXG/TEG是最高效策略
- 建议找目标部门内部员工直推，而非使用通用内推码

### 2.2 字节跳动（前雇主）

#### 招聘流程
- **轮次**：3轮技术面 + 1轮HR面
- **一面（45-65min）**：前端基础 + 手写代码 + 项目简述
- **二面（50-76min）**：技术深度 + 框架原理 + 算法实战 + 项目深挖
- **三面（60-105min）**：综合能力 + 系统设计 + 压力面可能
- **HR面（30-50min）**：离职原因 + 职业规划 + 薪资期望
- **2025年显著变化**：增加AI Native思维考察（AI在工作的应用、MCP/Agent模式）

#### 薪资结构
| 组件 | 详情 |
|------|------|
| Base月薪 | 2-2：35K-60K / 3-1：50K-80K |
| 年终奖 | M档+1.5月、M+档+2.5月、E档+3月（2025新政） |
| 期权 | 2-1少量、2-2较丰厚，未上市但内部回购（8折，2026年起55%可立即回购） |
| 标准薪制 | 15薪 |
| 总包参考（2-2/3-1） | 60万-130万/年 |
| 2025年底涨薪 | 奖金投入提升35%，调薪预算提升1.5倍 |

#### 隐形门槛
- **学历**：校招卡211/985，社招本科起步，更看重项目质量
- **年龄**：团队平均26-28岁，30+需证明"学习能力不输年轻人"
- **跳槽频次**：字节自身人才流动大，对跳槽频率容忍度较高
- **价值观**："始终创业"文化，面试中会直接考察对高强度工作的态度

#### HC情况（2025-2026）
- 在招职位771个，前端"有比较大的空缺"
- 飞书商业化不及预期（2024优化），但抖音电商/国际化/AI方向HC充足
- 2026提前批覆盖深圳、杭州、北京、上海等6个城市

#### 内推价值
- **高**：字节内推体系成熟，内推简历必定被HR查看
- 张三有字节前员工身份，联系前同事获取内推是最佳路径
- 注意：字节面试官风格"追根究底"，需准备源码级深度

- **年龄**：不敏感，团队年龄结构较均衡

- "摩天轮"内部轮岗计划成熟，适合长期发展

#### 薪资结构
| 组件 | 详情 |
|------|------|

| 股票归属（2025新规） | 15%/25%/30%/30%（4年），第一年即可归属15% |
| 总包参考（P7/P8） | 56万-180万/年 |

#### 隐形门槛

- **年龄**：P7对应3-5年经验，P8对应5-8年经验；30+在P7/P8正常

| 35-40岁 | 开始遇冷，但P8+架构岗仍紧缺 |
| 40+ | 除非技术专家/管理双通道，否则难度大增 |

> 张三约27-28岁（2019年毕业），处于前端黄金期，年龄零阻力。

### 5.3 跳槽频率门槛

| 频率 | 2025年实际接受度 |
|------|-----------------|
| 2-3年一跳 | 完全正常，零减分 |
| 1.5年一跳 | 需合理解释 |
| 1年一跳x2+ | 简历筛选可能被卡，但前端缺人时HR会放宽 |
| <1年多次 | 红线，除非强力内推 |

> 张三经历：Shopee 1.9年→字节 1.7年→腾讯 3年+（至今），前两段略短但有大厂跳槽的合理性（职业成长），无需担心。

2. **项目交付能力**：能说清"主导了什么、解决了什么、带来了什么业务结果"

## 七、背调合规指南

### 7.1 背调七大核查维度

1. **身份与学历**：学信网核实
2. **工作履历完整性**：入职/离职时间、职级、薪资、时间缝隙
3. **离职原因真实性**：主动离职 vs 被动淘汰 vs 重大失职
4. **工作表现**：前上级/同事评价、实际项目贡献
5. **商业道德**：竞业协议、带走的客户/资源/机密
6. **法律风险**：失信被执行人、劳动仲裁、诉讼记录
7. **名下公司**：挂靠股东/监事/法人（必须在背调启动前移除）

### 7.2 背调红线（直接取消Offer）

- 学历造假
- 工作履历造假（虚构公司、合并经历、延长在职时间 > 3个月）
- 伪造薪资流水/离职证明
- 隐瞒竞业协议
- 虚构项目成果

### 7.3 张三背调准备清单

- [ ] 确认Shopee/字节/腾讯三段经历的起止时间精确到月、无3个月以上偏差

- [ ] 检查名下无公司挂靠（董事/监事/法人）
- [ ] 准备好社保记录、离职证明扫描件
- [ ] 简历与背调简历内容完全一致
